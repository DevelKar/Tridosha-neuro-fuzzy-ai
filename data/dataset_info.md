Synthetic Ayurvedic Dataset for Tridosha Classification

The dataset used in this project is a synthetic dataset generated to simulate Ayurvedic health assessments based on the Tridosha system (Vata, Pitta, Kapha). It consists of input features such as age, sleep patterns, digestion strength, body temperature, stress levels, and dietary habits, which are key indicators in Ayurvedic diagnosis.

The output labels (Vata, Pitta, Kapha percentages and dominant dosha) are generated using a rule-based approach inspired by Ayurvedic principles. These rules model the relationship between lifestyle factors and dosha imbalances, ensuring that the dataset reflects realistic physiological patterns.

Synthetic data is used due to the lack of publicly available labeled Ayurvedic datasets, while still maintaining logical consistency and interpretability. This dataset enables the development and testing of machine learning and fuzzy logic models for personalized health analysis.
