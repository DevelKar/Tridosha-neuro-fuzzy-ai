# Tridosha-neuro-fuzzy-ai

# 🧠 Hybrid Neuro-Fuzzy Tridosha Prediction System

## 📌 Overview

The **Hybrid Neuro-Fuzzy Tridosha Prediction System** is an intelligent health analysis tool inspired by Ayurvedic principles. It predicts an individual’s **Vata, Pitta, and Kapha** composition based on lifestyle and physiological inputs such as sleep, digestion, stress, and diet.

The system combines:

* **Fuzzy Logic** → for rule-based Ayurvedic reasoning
* **Neural Learning (ML concept)** → for adaptability and improved predictions

This hybrid approach ensures the system is both **interpretable and data-driven**.

---

## ✨ Features

* 🔍 Predicts **Vata, Pitta, Kapha percentages**
* 🧬 Identifies **dominant dosha**
* 📖 Provides **explanation based on rules**
* 💡 Generates **personalized recommendations**
* ⚡ Hybrid system (Fuzzy + ML-ready)
* 🌐 Full-stack implementation (Frontend + Backend)

## 🏗️ Architecture

The system follows a hybrid layered architecture:

1. **Input Layer**

   * User inputs: age, sleep, digestion, body temperature, stress, diet

2. **Preprocessing**

   * Data validation and formatting
   * Encoding categorical values

3. **Fuzzy Logic Engine**

   * Applies rule base (`rules.json`)
   * Assigns scores to Vata, Pitta, Kapha

4. **Neural Learning Layer**

   * Uses synthetic dataset for learning patterns
   * Improves prediction accuracy and generalization

5. **Output Layer**

   * Dosha percentages
   * Dominant dosha
   * Explanation & recommendations

---

## ⚙️ How It Works

1. User fills out a health assessment form
2. Data is sent to backend API
3. Fuzzy rules are applied to calculate dosha scores
4. Scores are normalized into percentages
5. (Optional) ML model refines prediction
6. Final result is returned and displayed with:

   * Dosha distribution
   * Dominant dosha
   * Explanation
   * Recommendations
